# About the New Version of Wallet

> **Date:** February 20, 2023  
> **Category:** Notice

Wallet version `1.2.1` adds decentralized ETH, ERC20-USDT, and token functions.

Users should properly protect their wallet seed and private key when creating a new wallet, as they are the guarantee of user assets.

The next generation wallet version will include a new UI design in order to better support atomic exchange.

The update of the Apple version wallet has been delayed due to issues with the Apple system development site.

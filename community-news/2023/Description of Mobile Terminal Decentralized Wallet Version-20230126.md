# Description of Mobile Terminal Decentralized Wallet Version

> **Date:** January 26, 2023  
> **Category:** Notice

The next decentralized wallet version, version `1.2.1`, will add decentralized ETH, ERC20-USDT, and the ability to import ETH chain token functions.

It is expected to complete all functions and go online by the end of February 2023.

Version `1.2.1` of the wallet will be compatible with all previous versions.

# Announcement on the Completion of Atomic Switching Test Network

> **Date:** March 25, 2023  
> **Category:** Notice

The Judecoin team has completed the BTC-JUDE atomic exchange test on the test network.

In order to ensure that the software can be used on the main network without causing financial losses, a main network test will be conducted.

After confirming that it is safe and accurate, the main network wallet will be launched.

A new wallet version will be released after the main network test is completed.

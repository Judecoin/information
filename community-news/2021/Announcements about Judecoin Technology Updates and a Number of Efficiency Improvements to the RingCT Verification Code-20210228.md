# Announcements about Judecoin Technology Updates and a Number of Efficiency Improvements to the RingCT Verification Code

> **Date:** February 28, 2021  
> **Category:** Notice

Since its launch, Judecoin has been welcomed by the majority of community blockchain enthusiasts, who have made many contributions to Judecoin.

Through unremitting efforts, the Judecoin technical team and community enthusiasts have made the following updates and upgrades to Judecoin technology:

1. Greatly improved the speed of RingCT verification.

2. Cached transaction hashes for failed transactions with bad semantics.

3. Implemented initial intelligent mining on Linux only, and improved IBD (initial block download) and synchronization speed.

4. Added uptime to daemon state and added a fix for fluffy blocks.

5. Added support for multiple daemons using the same LMDB database.

6. Added major improvements to `libwallet_api`, allowing the GUI to use additional functionality and fixing ARMv8 build issues.

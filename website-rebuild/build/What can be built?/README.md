# What can be built?

This folder contains the design materials for the **What can be built?** page of the new Judecoin website.

This page is a content presentation page. The page text is based on the corresponding Judecoin information page, while this folder provides the UI design previews and standalone HTML reference.

## Content Source

The page content is based on:

[What can be built?](https://github.com/Judecoin/information/blob/main/What%20can%20be%20built%3F.md)

The information page is the source of truth for the page text.

## Notes

This page does not require a separate detailed requirements document.

The page implementation should follow the content from the What can be built? information page and use the uploaded design previews and HTML file as the visual reference.

The final implementation may be further refined based on the project’s technical standards and design decisions.

# Downloads

This folder contains the design materials for the **Downloads** page of the new Judecoin website.

This page is a software download presentation page. The page text and download references come from the existing Downloads page, while this folder provides the UI design previews and standalone HTML reference.

## Content Source

The page content is based on:

[Downloads](https://www.judecoin.info/downloads)

The existing Downloads page is the source of truth for the page text and download references.

## Files

- [Dark Theme Preview](./downloads-dark.png)  
  Dark theme design preview for the Downloads page.

- [Light Theme Preview](./downloads-light.png)  
  Light theme design preview for the Downloads page.

- [Standalone HTML Preview](./downloads.zip)  
  Contains a standalone `downloads.html` file for local browser preview.

## Notes

This page does not require a separate detailed requirements document.

The page implementation should follow the content from the existing Downloads page and use the uploaded design previews and HTML file as the visual reference.

The final implementation may be further refined based on the project’s technical standards and design decisions.

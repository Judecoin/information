# Service Nodes

This folder contains the design materials for the **Service Nodes** page of the new Judecoin website.

This page is a content presentation page. The page text is based on the corresponding Service Node setup, staking, restart, recovery, and CLI wallet operation documentation, while this folder provides the UI design previews and standalone HTML reference.

## Content Source

The page content is based on:

[Full Service Node Setup](https://github.com/Judecoin/wallet/blob/main/full-service-node-setup.md)

[Service Node Staking & Operations](https://github.com/Judecoin/pos-evolution/blob/main/service-node-staking.md)

[Service Node Restart Guide](https://github.com/Judecoin/pos-evolution/blob/main/service-node-restart.md)

[Service Node Recovery Using SN Private Key](https://github.com/Judecoin/pos-evolution/blob/main/service-node-recovery-with-sn-key.md)

[Linux CLI Wallet Setup Guide](https://github.com/Judecoin/wallet/blob/main/linux-cli-wallet-setup.md)

[Windows CLI Wallet Setup Guide](https://github.com/Judecoin/wallet/blob/main/windows-cli-wallet-setup.md)

[Linux CLI Wallet Recovery Guide](https://github.com/Judecoin/wallet/blob/main/linux-cli-wallet-recovery.md)

[Windows CLI Wallet Recovery Guide](https://github.com/Judecoin/wallet/blob/main/windows-cli-wallet-recovery.md)

These documentation pages are the source of truth for the page text.

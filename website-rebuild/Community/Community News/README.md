# Community News

This folder contains the design materials for the **Community News** page of the new Judecoin website.

This page is a content presentation page. The page text is based on the corresponding Judecoin Community News archive, while this folder provides the UI design previews and standalone HTML reference.

## Content Source

The page content is based on:

[Community News](https://github.com/Judecoin/information/tree/main/community-news)

The Community News archive is the source of truth for the page text.

This page does not require a separate detailed requirements document.

The page implementation should follow the content from the Community News archive and use the uploaded design previews and HTML file as the visual reference.

The final implementation may be further refined based on the project’s technical standards and design decisions.

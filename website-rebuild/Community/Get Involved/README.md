# Get Involved

This folder contains the design materials for the **Get Involved** page of the new Judecoin website.

This page is a content presentation page. The page text comes from the corresponding information page, while this folder provides the UI design previews and standalone HTML reference.

## Content Source

The page content is based on:

[Get involved](https://github.com/Judecoin/information/blob/main/Get%20involved.md)

The information page is the source of truth for the page text.

This page does not require a separate detailed requirements document.

The page implementation should follow the content from the information page and use the uploaded design previews and HTML file as the visual reference.

The final implementation may be further refined based on the project’s technical standards and design decisions.

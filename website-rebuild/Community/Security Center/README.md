# Security Center

This folder contains the design materials for the **Security Center** page of the new Judecoin website.

This page is a content presentation page. The page text is based on the corresponding Judecoin Service Node security, backup, and status documentation, while this folder provides the UI design previews and standalone HTML reference.

## Content Source

The page content is based on:

[Judecoin Service Node Security and Backup Checklist](https://github.com/Judecoin/pos-evolution/blob/main/service-node-security-backup.md)

[Judecoin Service Node Status and Common FAQ](https://github.com/Judecoin/pos-evolution/blob/main/service-node-status-faq.md)

These documentation pages are the source of truth for the page text.

This page does not require a separate detailed requirements document.

The page implementation should follow the content from the Judecoin Service Node security, backup, and status documentation and use the uploaded design previews and HTML file as the visual reference.

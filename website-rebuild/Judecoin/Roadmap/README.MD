# Roadmap Page Rebuild

This folder contains the Roadmap page rebuild materials for the new Judecoin official website.

The Roadmap page presents Judecoin’s current-year development items, future planned work, and completed foundations from previous years. It defines the Roadmap content structure, year switching logic, status display, default open and collapsed states, and expand / collapse interaction.

## Files

- [Roadmap Page Requirements](./Roadmap-requirements.md)  
  Defines the Roadmap page content structure, current-year logic, completed foundations logic, future plans logic, status display, default open and collapsed behavior, and expand / collapse interaction.

- [Dark Theme Roadmap Default View](./roadmap-dark-default.png)  
  Shows the Roadmap page in dark theme with the default open state.

- [Dark Theme Roadmap Collapsed View](./roadmap-dark-collapsed.png)  
  Shows the Roadmap page in dark theme with the collapsed state.

- [Light Theme Roadmap Default View](./roadmap-light-default.png)  
  Shows the Roadmap page in light theme with the default open state.

- [Light Theme Roadmap Collapsed View](./roadmap-light-collapsed.png)  
  Shows the Roadmap page in light theme with the collapsed state.

- [Standalone Roadmap HTML Preview](./Roadmap.html.zip)  
  Contains a standalone `Roadmap.html` file for local browser preview. Download and extract the ZIP package, then open `Roadmap.html` in a browser to review the Roadmap layout, status navigation, year cards, theme switching, and expand / collapse interaction.

## Roadmap Reference Materials

The Roadmap requirements document, page screenshots, and standalone HTML preview are provided together as the reference materials for the Roadmap page rebuild.

The screenshots show the Roadmap page structure, status navigation, current-year Roadmap area, Future Plans area, Completed Foundations area, default open state, collapsed state, and visual presentation in both dark and light themes.

The standalone HTML preview shows the Roadmap page flow, theme switching, status navigation, expandable Roadmap cards, and interaction behavior in a local browser.

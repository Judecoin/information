# Next Fork Wallet Version Upgrade Order

> **Date:** August 8, 2025  
> **Category:** Notice

The next fork will prioritize upgrading the CLI wallet, which will operate all Service Node functions.

The mobile wallet will be upgraded next, followed by the desktop wallet.

This is the current order for upgrading the various wallet versions.

The following are notes for node operators:

1. Required stake amount to become a node.

2. Number of operators and contributors in a shared node, and their stake ratios.

3. Node operation fees.

4. Node reward distribution.

5. Four node states: deregister, decommission, recommission, and IP change penalty (`ip_change_penalty`).

6. Minimum number of nodes required for the Service Node network to maintain health.

7. Minimum quorum required for node status changes.

8. Minimum vote required for node status changes.

9. Node IP change detection window: `24 hours`.

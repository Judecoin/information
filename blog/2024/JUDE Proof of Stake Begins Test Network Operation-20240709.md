# JUDE Proof of Stake Begins Test Network Operation

> **Date:** July 9, 2024  
> **Category:** Notice

JUDE Proof of Stake (PoS) plans to start testing on the test network.

The full plan will be implemented on the main network after the test network completes various functions.

JUDE Block Browser V2.0 will be used after the main network implements Proof of Stake.

Website: https://www.judecoin.org

Mobile wallet voting will be used for the next integrated decentralized wallet.

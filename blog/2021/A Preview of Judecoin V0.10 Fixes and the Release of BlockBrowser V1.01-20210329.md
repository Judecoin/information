# A Preview of Judecoin V0.10 Fixes and the Release of BlockBrowser V1.01

> **Date:** March 29, 2021  
> **Category:** Notice

This is the requisite point version of Judecoin V0.10 "Wolfram Warp" and is highly recommended because it includes a consensus change fix for the RingCT implementation and various other bug fixes.

Some of the highlights of this release are:

- Support for major GUI changes.
- Added full support for "soft blocks," a propagation improvement similar to compact blocks in Bitcoin Core.
- Added a dynamic billing system.
- Extended data stored in the wallet cache, including GUI address books.
- Thorough overhaul of the thread system.
- Optimized wallet blockchain refresh mechanism.
- Switched to a dynamic dust threshold system.
- Added a command to calculate the entire Coinbase.
- Eliminated the old quick-exit mechanism and addressed related issues.
- Improved and revised the cold wallet transaction signature mechanism.
- Overhauled scan immiscible implementation.
- Fixed FreeBSD builds.

With various feature fixes and improvements, BlockBrowser version V1.01 is expected to be available in late April 2021. Stay tuned.

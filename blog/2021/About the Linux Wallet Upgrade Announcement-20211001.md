# About the Linux Wallet Upgrade Announcement

> **Date:** October 1, 2021  
> **Category:** Notice

With the improvement of the Judecoin community and technology, the Judecoin wallet has been upgraded.

The current version is:

`Linux-0.1.2.0`

After the new wallet version is released, users can use the mnemonic seed words to restore the wallet address in the new version.

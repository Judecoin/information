# Announcement about Judecoin-0.10.0-Wallet-Linux (64)

> **Date:** April 27, 2021  
> **Category:** Release

## Summary

Due to the performance degradation of 32-bit Windows, some environments may not be supported.

In order to focus on the functions required by the initial version, support is being completed for 64-bit Windows, FreeBSD, DragonFly BSD, armv7 Linux, and armv8 Linux devices.

Judecoin-0.10.0-Wallet-Linux (64) is expected to be released in June.

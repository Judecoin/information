# Announcement on the Release of the New Wallet and the Launch of the Windows Wallet

> **Date:** November 13, 2021  
> **Category:** Notice

Judecoin has always attached importance to on-chain infrastructure and ecosystem construction.

After the Linux wallet is upgraded to the new version `Linux 0.1.2.1` and the Windows wallet is released, Judecoin will launch a more user-friendly mobile wallet in the future.

Before the new application version is launched, all old versions will be updated to the latest version, and new features will come online.

In the future, Judecoin will continue to focus on the construction of on-chain infrastructure.

In order to break monopoly, achieve true decentralization, and better serve basic users in the community, the GUI wallet will prohibit large-amount transfer transactions.

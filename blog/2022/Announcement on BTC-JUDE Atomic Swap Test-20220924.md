# Announcement on BTC-JUDE Atomic Swap Test

> **Date:** September 24, 2022  
> **Category:** Notice

The wallet is expected to test atomic swaps on the testnet in December.

Once the test is considered generally stable enough, it will be switched to mainnet.

The Judecoin community will continue to develop and maintain APIs that allow clients to easily discover exchange providers.

Swap providers are peers that users can connect with to complete the atomic swap function. The API indexes them and provides additional data such as uptime and age.

This API is publicly accessible and can be used by other services.

Centralized peer discovery is provided through an unstoppable exchange API. A database of exchange providers is actively maintained, allowing anyone to retrieve a list of available exchange providers.

The community also operates a rendezvous mechanism.

Rendezvous is a lightweight protocol for generalized peer discovery that allows peer discovery in a decentralized manner.

Through this mechanism, exchange providers can make themselves known to users, and users can find exchange providers they want to exchange with.

Users may also view, merge, and possibly submit pull requests to the Jude-BTC exchange repository.

This proposal is mainly used to continue developing the wallet, not only to maintain the Jude-BTC exchange project.

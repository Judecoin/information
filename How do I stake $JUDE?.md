# How do I stake JUDE?

JUDE staking is done through Judecoin Service Nodes.

Judecoin Service Nodes provide underlying support for the Judecoin network. They are powered by rewarded nodes to enable network functionality, scalability, and a decentralized technology stack.

To operate a Judecoin Service Node, operators can lock up a minimum of 23,600 JUDE. The stake transaction is submitted to the network, and after a short delay, the service node is registered and begins contributing to the network.

In return for their services, Judecoin Service Node operators receive a portion of the average block reward from every block mined on the Judecoin blockchain.

## Service Node Staking

A Judecoin Service Node becomes active on the network when its owner stakes the required amount of JUDE and submits a registration transaction.

Once accepted by the network, the Judecoin Service Node becomes eligible to receive block rewards.

Running a Judecoin Service Node is the perfect way to participate in the Judecoin privacy network while earning rewards.

## Shared Service Nodes

Multiple participants can stake into a shared service node by combining smaller stakes to meet the 23,600 JUDE staking requirement.

Block rewards can be automatically distributed to participants in proportion to each participant's stake contribution.

Users can stake into a shared service node directly from the wallet without using the command line.

## How to Participate in a Shared Service Node

To participate in a shared service node:

1. Open the Judecoin wallet.
2. Navigate to the Service Nodes tab.
3. Enter the public key of the service node you are contributing to.
4. Enter the amount of JUDE you want to stake.
5. Click the Stake button.

After staking, rewards can begin flowing into the Judecoin wallet according to the network rules and the participant's stake contribution.

If users need to find a shared Judecoin Service Node to contribute to, they can use the Service Nodes section of Judecoin Blockchain Explorer II.

## Running Your Own Judecoin Service Node

Users who want to operate their own Judecoin Service Node should prepare a suitable server environment, run the latest Judecoin service node software, keep the node online, and complete the service node registration process.

A typical Judecoin Service Node setup may include:

- a supported Linux server environment;
- sufficient storage, memory, network connectivity, and monthly traffic;
- open and reachable service node ports;
- the latest Judecoin service node binaries;
- a fully synchronized Judecoin daemon;
- a Judecoin wallet holding the required staking amount;
- service node registration from the wallet;
- regular status checks and key backups.
